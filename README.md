# 🚀 DevOps Lab Record

## 📌 Overview

This repository contains a well-structured record of **10 days of DevOps practical work** completed during lab sessions. It demonstrates hands-on experience with essential DevOps tools, commands, and workflows.

* Creating repositories

* Understanding containers vs virtual machines
